CDT-plusplus
============

Causal Dynamical Triangulations in C++ using the [Computational Geometry Algorithms Library][1] and compiled with [LLVM][2] and [CMake][3]

- [ ] Simplex-class
- [ ] Tests using [Google Test][4]
- [ ] Initialize
- [ ] Add curvature
- [ ] Ergodic moves
- [ ] Metropolis algorithm
- [ ] Shortest path algorithm
